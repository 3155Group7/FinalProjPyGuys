To run code run qLife.py
this can be done in command line with code python qLife.py as long as flask is intalled if need install flask using pip install flask
open webpage at localhost:5000/Home

After installing the dependencies listed below run the following commands if using your terminal.

BASH:

$ export FLASK_APP=qLife
$ flask run

CMD:

> set FLASK_APP=qLife
> flask run

PowerShell:

> $env:FLASK_APP = "qLife"
> flask run
